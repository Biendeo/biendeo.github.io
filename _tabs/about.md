## Who is this guy?
I'm a fellow Australian who enjoys video games and music. I've been running a [YouTube channel](https://youtube.com/Biendeo) for a while where I commentate playthroughs of various video games I've enjoyed (and some not so much enjoyed). I've also been live streaming these playthroughs on [my Twitch channel](https://twitch.tv/Biendeo), and upload all the VODs to YouTube so you can view them back.

I also do programming for a living, and I dabble in a bit as a hobby from time to time. You might see some projects show up from time to time here!

## So what do you post here?
On this blog I'll be posting either things I'm working on, opinions, or just generally writing about things I've experienced and done. I want to put some thoughts on things I do down in written form so I can remind myself about it later, and perhaps you might enjoy what I write here!
